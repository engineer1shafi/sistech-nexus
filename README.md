# SISTECH NEXUS

Enterprise Network Operations Platform.

## Core Features

- SNMP Monitoring
- ICMP Monitoring
- Device Inventory
- Live Network Topology
- NetFlow / IPFIX
- Syslog
- Alert Manager
- RBAC
- Multi-Tenant Architecture
- Docker Deployment

## Technology Stack

- Backend: Python, FastAPI
- Frontend: React, TypeScript
- Database: PostgreSQL, TimescaleDB
- Cache: Redis
- Deployment: Docker Compose

## Status

Development started with v0.0.1-dev.
