# AGENTS

## Project guidance for contributors and coding agents

- Follow the existing repository structure in the backend and frontend folders.
- Keep changes scoped to the relevant feature area and avoid unrelated rewrites.
- Prefer the established service-layer pattern in the backend and the existing React/TypeScript structure in the frontend.
- For backend work, keep FastAPI routes thin and place business logic in services and repositories.
- For frontend work, keep UI state and API calls consistent with the existing component and service patterns.
- Verify changes with the relevant local checks before finishing.
